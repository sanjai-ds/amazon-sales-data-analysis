# 📊 Amazon Sales Data Analysis & Visualization

---

## 📌 Project Overview
This project focuses on analyzing Amazon sales data to uncover meaningful insights about customer behavior, product performance, and revenue trends. The analysis helps in understanding how different factors influence sales and supports data-driven business decisions.

---

## 🎯 Problem Statement
The objective of this project is to analyze Amazon sales data to:
- Understand customer purchasing behavior  
- Identify top-performing product categories  
- Analyze revenue trends across different regions  
- Evaluate the impact of ratings and payment methods on sales  
- Provide actionable insights to improve business performance  

---

## 🛠️ Tools & Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook
